A constante evolução da internet transforma o modo como acessamos informações online e de que maneira interagimos com elas. Assim, as interfaces se tornaram mais complexas para atender às necessidades de cada grupo de visitantes.

Por tanto o Desenvolvedor Web Front-end é um profissional vital que encarrega-se de projetar estruturas web, sempre pensando na experiência do usuário.

Objetivo

Ser um Desenvolvedor Web Front-end que mais do que construir sites se preocupa com a experiência do usuário com um produto ou serviço, que hoje em dia acontece cada vez mais em ambientes online, trabalhando com profissionalismo para desenvolver sistemas com interfaces enxutas, eficientes e bem projetadas.

Programar em HTML, CSS e JavaScript, produzindo layouts de produtos digitais e criando interatividade entre os elementos de uma página ou de um aplicativo. Procurando não estar inserido num único time, mas transitando entre as áreas de design, de Desenvolvimento Web Back-end, infraestrutura e de QA.
